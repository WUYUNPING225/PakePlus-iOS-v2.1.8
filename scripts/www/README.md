# 批量邮件发送平台

一个基于HTML、CSS和JavaScript的批量邮件发送工具，支持个性化邮件内容、附件发送和定时发送功能。

## 功能特点

- **邮箱配置**：支持配置各种SMTP服务器，包括Gmail、QQ邮箱、163邮箱等
- **收件人管理**：支持CSV文件导入和手动添加收件人
- **邮件编辑**：支持个性化邮件内容，使用{姓名}占位符实现自动替换
- **附件管理**：支持上传多个附件文件
- **发送控制**：支持设置发送间隔时间，默认1分钟发送一封
- **本地存储**：使用浏览器本地存储保存配置和数据，避免数据丢失
- **响应式设计**：适配桌面和移动设备

## 如何使用

1. 打开`index.html`文件
2. 在"邮箱配置"页面填写您的发件人邮箱和SMTP服务器信息
3. 在"收件人管理"页面添加或导入收件人列表
4. 在"邮件编辑"页面编写邮件内容，可使用{姓名}占位符
5. 在"附件管理"页面上传需要发送的附件
6. 在"发送控制"页面设置发送间隔并开始发送

## 邮箱配置说明

使用前需要在邮箱设置中开启SMTP服务并获取授权码（不是邮箱密码）。

常见邮箱SMTP设置：

| 邮箱服务 | SMTP服务器 | 端口 | 加密方式 |
|---------|-----------|------|---------|
| Gmail | smtp.gmail.com | 587 | TLS |
| QQ邮箱 | smtp.qq.com | 587 | SSL |
| 163邮箱 | smtp.163.com | 465 | SSL |
| Outlook | smtp-mail.outlook.com | 587 | TLS |

## CSV文件格式

导入的CSV文件应包含姓名和邮箱地址列，例如：

```csv
姓名,邮箱
张三,zhangsan@example.com
李四,lisi@example.com
```

## 注意事项

1. 请确保您的浏览器支持现代JavaScript特性
2. 由于浏览器限制，附件总大小不应超过25MB
3. 发送大量邮件时，请遵守邮件服务器的发送限制
4. 建议先发送测试邮件确认配置正确
5. 本工具在前端运行，不会将您的邮箱信息上传到任何服务器

## 技术栈

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript (ES6+)
- SMTP.js (用于发送邮件)
- PapaParse (用于解析CSV文件)
